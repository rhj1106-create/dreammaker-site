═══════════════════════════════════════════════════════
  배포 방법 (가장 쉬운 순서)
═══════════════════════════════════════════════════════

[방법 1] Vercel — 추천 (무료, 5분)
─────────────────────────────────
1. https://vercel.com 접속 → Sign Up
2. 가입 후 "Add New Project" → "Browse" 클릭
3. 이 deploy 폴더 통째로 드래그앤드롭
4. "Deploy" 버튼 클릭
5. 끝나면 임시 주소 받음 (예: dreammaker-xxxx.vercel.app)
6. 임시 주소로 작동 확인
7. Settings → Domains → "dreammaker.kr" 추가
8. 안내되는 DNS 값(A 레코드 / CNAME)을 도메인 등록처에 입력


[방법 2] Netlify — 가장 간단 (무료)
─────────────────────────────────
1. https://app.netlify.com 접속 → 가입
2. 메인 화면에서 deploy 폴더를 화면에 드래그앤드롭
3. 즉시 배포 완료, 임시 주소 받음
4. Domain settings에서 dreammaker.kr 추가
5. 안내되는 DNS 값을 도메인 등록처에 입력


[방법 3] 카페24 / 가비아 등 한국 호스팅
─────────────────────────────────
1. 카페24(www.cafe24.com) 등에서 웹호스팅 신청
2. FTP 접속 정보 수령
3. FileZilla 등 FTP 프로그램으로 deploy 폴더 내용 업로드
4. 도메인 연결 메뉴에서 dreammaker.kr 등록


═══════════════════════════════════════════════════════
  도메인 DNS 설정 (Vercel/Netlify 사용 시)
═══════════════════════════════════════════════════════

dreammaker.kr 도메인을 산 사이트(가비아/후이즈/카페24 등)에서:

[A 레코드]
  호스트:  @ (또는 빈칸, 또는 dreammaker.kr)
  값:      Vercel/Netlify가 안내하는 IP 주소

[CNAME 레코드]
  호스트:  www
  값:      cname.vercel-dns.com (Vercel 사용시)
           또는 호스팅이 안내하는 값

설정 후 10분~24시간 내 도메인 활성화
